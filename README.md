# HytaleDev

HytaleDev is an IntelliJ IDEA plugin that adds a “Hytale” entry to the New Project wizard and generates a ready-to-run Hytale plugin project from templates.

## Modules

- `core`: Pure Kotlin library (no IntelliJ dependencies)
- `idea-plugin`: IntelliJ Platform plugin (wizard UI, settings, actions)
- `cli` (optional): Thin wrapper over `core`
